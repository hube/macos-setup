# macOS Setup

A set of scripts to help ease the setup of a new Mac. This repo includes
.dotfiles used to configure my machine. [Learn more about dotfiles][1].
Previously, this repo contained a number of development tools. These have been
removed in favor of using [Development Containers][2] instead

# Usage

`source setup.sh`

# Post installation

## GitHub configuration

1. An SSH keypair was generated by the `4.1-setup-ssh-key.sh` script. Add the
  public key to GitHub so that it recognizes git commits signed with the keypair
2. Generate a [personal access token][3] and update the `GH_TOKEN` environment
  variable in `~/.oh-my-zsh/custom/devcontainer.zsh` so that devcontainers can
  use the `gh` CLI tool

[1]: http://dotfiles.github.io/
[2]: https://containers.dev/
[3]: https://docs.github.com/en/authentication/keeping-your-account-and-data-secure/managing-your-personal-access-tokens
