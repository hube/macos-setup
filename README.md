macOS Setup
==========
A set of scripts to help ease the setup of a new Mac. This repo includes
.dotfiles used to configure my machine. [Learn more about dotfiles][1].
Previously, this repo contained a number of development tools. These have been
removed in favor of using [Development Containers][2] instead

Usage
=====
`source setup.sh`

[1]: http://dotfiles.github.io/
[2]: https://containers.dev/
